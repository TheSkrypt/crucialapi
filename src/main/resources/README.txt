=========================
=== FOR SERVER OWNERS ===
=========================
Congratulations, you successfuly installed Crucial API.

You most likely installed the API because another plugin requires it.
In case you remove all plugins that require this API, you should remove it from your plugins as it will
be of no more use to you and only take up your space and performance.

======================
=== FOR DEVELOPERS ===
======================
This is an open source API, if you want to have a look at the source or wish to contribute,
you are free to do so under https://github.com/TheSkrypt/crucialapi

The documentation, wiki and all other important information can be found there too.
